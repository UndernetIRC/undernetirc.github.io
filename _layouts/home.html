Welcome

The Undernet Coder Committee is a group of volunteers responsible for the design and maintenance of Undernet's core software services.

Projects

IRCu

IRCu is the core IRC daemon that all Undernet servers and hubs are running.

GNUWorld

GNUWorld is the set of IRC services that power Undernet's X, C and other service bots.

CService-Web

CService-Web is the codebase that powers Undernet's live CService web interface and user registration system.

IAuthd-c

IAuthd-C is the reference implementation for Undernet's IAuth protocol that enables features like pre-connection security scanning, login-on-connect and more.
